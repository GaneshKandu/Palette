# Palette

Palette is PHP Based Small Site Builder

## Getting Started

## Prerequirement

* PHP 5.3+

### Extension

* gd
* zip

## Installation

Create Folder

* thumbs/
* sites/

Give a Writable Permission on
* config/
* thumbs/
* sites/
* logs/error.log
* .htaccess

and open install.php in you Browser
